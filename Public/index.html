NC News API Documentation

GET /api

serves up a json representation of all the available endpoints of the api

GET /api/topics

serves an array of all topics

Example Response:

{
  "topics": [
    { 
      "slug": "football", 
      "description": "Footie!" 
    }
  ]
}

GET /api/articles

serves an array of all articles

Example Response:

{
  "articles": [
    {
      "title": "The Lord of the Rings",
      "topic": "fiction",
      "author": "John Ronald Reuel Tolkien",
      "created_at": "2018-05-30T15:59:13.341Z",
      "votes": 0,
      "comment_count": 6
    }
  ]
}

GET /api/users

serves an array of all users

Example Response:

{
  "users": [
    {
      "username": "The White Wizard",
      "name": "Gandalf",
      "avatar_url": "https://example.com/avatar.jpg"
    }
  ]
}

GET /api/articles/:article_id

serves a single article object by article ID

Example Response:

{
  "article": {
    "article_id": 2,
    "author": "John Ronald Reuel Tolkien",
    "title": "The Lord of the Rings",
    "body": "Text from the article..",
    "topic": "fiction",
    "created_at": "2018-05-30T15:59:13.341Z",
    "votes": 0,
    "article_img_url": "https://example.com/image.jpg"
  }
}

GET /api/articles/:article_id/comments

serves an array of comments for the given article ID, sorted by created_at (most recent first)

Example Response:

{
  "comments": [
    {
      "comment_id": 5,
      "votes": 0,
      "created_at": "2018-05-30T15:59:13.341Z",
      "author": "Gandalf",
      "body": "YOU SHALL NOT PASS!!",
      "article_id": 1
    }
  ]
}

POST /api/articles/:article_id/comments

adds a new comment to the specified article and returns the posted comment

Example Request:

{
  "username": "Gandalf",
  "body": "Not all those who wander are lost"
}

Example Response:

{
  "comment": {
    "comment_id": 19,
    "article_id": 3,
    "author": "Gandalf",
    "body": "Not all those who wander are lost",
    "votes": 0,
    "created_at": "2018-05-30T15:59:13.341Z"
  }
}

PATCH /api/articles/:article_id

updates the votes on an article by the specified amount and returns the updated article

Example Request:

{
  "inc_votes": 5
}

Example Response:

{
  "article": {
    "article_id": 2,
    "author": "John Ronald Reuel Tolkien",
    "title": "The Lord of the Rings",
    "body": "Text from the article..",
    "topic": "cooking",
    "created_at": "2018-05-30T15:59:13.341Z",
    "votes": 5,
    "article_img_url": "https://example.com/image.jpg"
  }
}

PATCH /api/comments/:comment_id

updates the votes on a comment by the specified amount and returns the updated comment

Example Request:

{
  "inc_votes": -2
}

Example Response:

{
  "comment": {
    "comment_id": 5,
    "article_id": 1,
    "author": "Gandalf",
    "body": "YOU SHALL NOT PASS!!",
    "votes": -2,
    "created_at": "2018-05-30T15:59:13.341Z"
  }
}

DELETE /api/comments/:comment_id

deletes the comment with the given comment_id and responds with no content

Example Response:

{}

GET /api/users/:username

Retrieves a user object by their username

Example Request:

{}

Example Response:

{
  "user": {
    "username": "jessjelly",
    "name": "Jess Jelly",
    "avatar_url": "https://example.com/avatar.jpg"
  }
}

POST /api/users

Creates a new user with a unique username, name, and avatar URL. Responds with the newly created user object.

Example Request:

{
  "username": "newuser123",
  "name": "New User",
  "avatar_url": "https://example.com/newuser-avatar.jpg"
}

Example Response:

{
  "user": {
    "username": "newuser123",
    "name": "New User",
    "avatar_url": "https://example.com/newuser-avatar.jpg"
  }
}
